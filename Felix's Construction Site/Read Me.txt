Play and Clock
Felix's Construction Site

This game is a take on the Game and Watch Tabletop LCD games. Created by Sergio Meyer. Not for sale or profit.

[Controls]
Ctrl = Superpower
Left Arrow = Move left
Right Arrow = Move right
1 = Start Game
1 + 2 = Exit Game
Esc = Exit Game
F12 = Toggle fullscreen

[Settings]
Settings can be adjust in the settings.ini file. They allow you to adjust difficulty and enable Arduino functions.
